# snkmk

## Что делает этот код?
Он создает файл output.txt в который сохраняет простой код на python 

### Установка
```
git clone https://github.com/gogulya/snkmk.git
cd snkmk
conda activate snakemake-env
snakemake --cores 1
type output.txt
```
## Выходные данные 
На терминале должно высветиться: 
```
this code is working correctly
```
